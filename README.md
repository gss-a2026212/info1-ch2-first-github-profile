# 🚀 ミッション：AIとつくる自己紹介Webページ

本日のミッションは，クラウド開発環境（GitHub Codespaces）を起動し，生成AI（Copilot等）と協力して簡単な「自己紹介Webページ」を作成することです．
完成したら，AIとの対話ログをJSONファイルとしてエクスポートし，すべてのファイルの変更履歴をセーブ（コミット）して，提出（プッシュ）してください．

---

## 📝 課題要件

### 1．自己紹介のHTMLを作成する（構造化）
`index.html` の `<body>` タグの中に，以下の要素を含む自己紹介コードを記述してください．
- **見出し1 (`<h1>`)**: あなたの名前またはニックネーム
- **見出し2 (`<h2>`)**: 「私の好きなこと」や「部活・趣味」などの大見出し
- **段落 (`<p>`)**: 詳しい自己紹介文
- **リスト (`<ul>` & `<li>`)**: 好きな食べ物やアーティストなど，3つ以上のリスト項目

### 2．自己紹介のCSSを追加する（装飾）
`style.css` に，Webページの見た目を整える以下のCSSコードを追加してください．
- 全体の背景色（`background-color`）を初期の白から変更する．
- 文字の色（`color`）や，フォントサイズを調整する．
- 文字の周囲に余白（`padding` や `margin`）を設定する．

---

## 🤖 生成AI（Copilot）の活用ルール

本日の演習から，AI（Copilot Chat）をコーディングの相棒として使うことができます．ただし，以下のルールを必ず守ってください．

1. **「丸投げ」は禁止です：** AIにコードを作ってもらったら，そのまま貼り付けて終わりにしてはいけません．
2. **「説明させる」質問をすること：** AIが提示したHTMLやCSSのコードに対し，必ず **「このコードに書いてあるタグやプロパティの意味を説明してください」** と追加で質問し，説明文を読んで理解した上でコードを適用してください．
3. **対話は2往復以上行うこと：** 最低でも2回以上，AIに質問・相談を投げてください．

---

## 🛠️ 操作手順（Step-by-Step）

### Step 1：リポジトリの作成とCodespacesの起動
1. Google Classroomに提示された課題のリンクをクリックし，アサインメントを受け入れます．
2. 生成された自分専用のリポジトリページを開き，緑色の **`Code`** ボタン ➔ **`Codespaces`** タブ ➔ **`Create codespace on main`** をクリックします．
3. ブラウザ上に開発画面（VS Code）が起動するまで待ちます（約1〜2分かかります）．

### Step 2：AIに相談してコードを作成する
1. 画面上のAIチャット（Copilot Chat）アイコンをクリックしてチャットパネルを開きます．
2. AIに最初の質問を投げます．
   * *プロンプト例:* 「HTMLでシンプルな自己紹介のコードを作りたいです．見出しタグと段落タグを使ったテンプレートのコードを作成してください」
3. AIがコードを作成したら，必ず2回目の質問を投げます．
   * *プロンプト例:* 「提案してくれたコードの中にあるタグ（`<h1>` や `<ul>` など）の意味を詳しく説明してください」
4. 説明を理解したら，コードを `index.html` にコピー＆ペーストし，名前や自己紹介の内容を自分の情報に書き換えて保存（`Ctrl + S` / `Cmd + S`）します．
5. 同様に，CSS（背景色や文字色の変更方法など）についてもAIに相談し，得られたCSSコードを `style.css` に適用して保存します．
6. 右下のプレビュー（Live Preview等）を起動し，正しく表示されているか画面を確認します．

### Step 3：AIとの対話ログをエクスポートする
1. AIチャットパネルの右上にある **`…`（メニュー）** をクリックします．
2. 表示されたメニューから **`チャットのエクスポート（Export Chat）`** を選択します．
3. 保存場所をいま開いているワークスペース（リポジトリのフォルダ）に指定します．
4. ファイル名を **`copilot-chat-history.json`** としてそのまま保存します．

### Step 4：コミットとプッシュ（提出）
1. 左側メニューの **`ソース管理（ブランチのアイコン）`** を開きます．
2. 変更のあったファイル（`index.html`，`style.css`，`copilot-chat-history.json`）の横にある **`＋`** ボタンをクリックして「ステージ」します．
3. 上部のメッセージ入力欄に **`setup environment and ai chat`** と入力します．
4. **`コミット（チェックマーク）`** ボタンをクリックします（セーブ完了）．
5. 青色の **`変更の同期`** （またはプッシュ）ボタンをクリックします（送信完了）．

### Step 5：提出の確認
1. ブラウザで元のGitHubリポジトリページに戻り，リロードします．
2. ファイルリストに `copilot-chat-history.json` が追加されており，コミット履歴に「setup environment and ai chat」が記録されていることを確認します．
3. 確認できたら，そのリポジトリのURLをGoogle Classroomの提出欄に貼り付けて，提出を完了させてください．
